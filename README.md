# BlitzCheckers ⚡ — шашки нового поколения

> **Не просто доска.** Тактические задачи, AI Coach, блиц-дуэли, достижения, 4 скина доски и P2P-мультиплеер — в одном продукте.

## 🎯 Уникальное позиционирование

**BlitzCheckers** — единственная платформа, которая совмещает:

| Фича | Зачем это конкурентам сложно скопировать |
|------|----------------------------------------|
| **Тактический режим** | 5 curated-задач на комбинации (двойные/тройные взятия, дамка) |
| **Челлендж дня** | Новая позиция каждый день — возврат в приложение |
| **AI Coach** | Разбор пропущенных взятий и неполных серий после партии |
| **COMBO xN** | Визуальный фидбек за многофигурные взятия + звук |
| **Повтор партии** | Слайдер по ходам после игры |
| **4 скина доски** | Классика, Неон, Мрамор, Полночь (Pro) |
| **8 достижений** | Геймификация и удержание |
| **Zen-режим** | Только доска — для фокуса |
| **Ghost trail** | Подсветка последнего хода и съеденных клеток |

## Функций

- ✅ Мультиплеер по ссылке (PeerJS WebRTC)
- ✅ AI Coach с разбором ходов
- ✅ Глобальный лидерборд по городам
- ✅ Ниша: **блиц 3 мин + тактическая тренировка**
- ✅ Upgrade to Pro + скины (демо Stripe-ready)
- ✅ Анимации (Framer Motion), confetti, звук Web Audio
- ✅ Адаптивный дизайн
